Рядкові функції MySQL

Строкові функції повертають NULL, якщо довжина результату виявляється більше, ніж вказано в значенні серверного параметра max_allowed_packet.

Функція Опис
ASCII Повертає код числа, що представляє конкретний символ
CHAR_LENGTH Повертає довжину зазначеного рядка (в символах)
CHARACTER_LENGTH Повертає довжину зазначеного рядка (в символах)
CONCAT Об'єднує два або більше виразів разом
CONCAT_WS Об'єднує два або більше виразів разом і додає роздільник між ними
FIELD Повертає положення значення в списку значень
FIND_IN_SET Повертає положення рядка в списку рядків
FORMAT Форматує число як формат "#, # # #. # #", Округляючи його до певного числа десяткових розрядів
INSERT вставляє підрядок в рядок в заданій позиції для певного числа символів
INSTR Повертає позицію першого входження рядка в інший рядок
LCASE Перетворює рядок в нижній регістр
LEFT Витягує подстроку з рядка (починаючи з лівого)
LENGTH Об'єднує два або більше виразів разом
LOCATE Повертає позицію першого входження підрядка в рядок
LOWER Перетворює рядок в нижній регістр
LPAD Повертає рядок, яка заповнена з заданим рядком до певної довжини
LTRIM Видалення початкових пробілів з рядка
MID Витягує подстроку з рядка (починаючи з будь-якої позиції)
POSITION Повертає позицію першого входження підрядка в рядок
REPEAT Повторює рядок вказану кількість разів
REPLACE Замінює всі входження зазначеного рядка

Цифрові функції MySQL

Цифрові функції слугують для роботи із числами та виразами

Функція Опис
ABS Повертає абсолютне значення числа
ACOS Повертає синус дуги числа
ASIN Повертає тангенс дуги числа або тангенса дуги n і m
ATAN Повертає тангенс дуги n і m
ATAN2 Об'єднує два або більше виразів разом і додає роздільник між ними
AVG Повертає середнє значення виразу
CEIL Повертає найменше ціле значення, що перевищує або дорівнює кількості
CEILING Повертає найменше ціле значення, що перевищує або дорівнює кількості
COS Повертає косинус числа
COT Повертає котангенс числа
COUNT Повертає кількість записів в запиті SELECT
DEGREES Перетворює значення радіан в градуси
DIV Використовується для цілочисельного ділення
EXP Повертає значення e, зведене в силу числа
FLOOR Повертає найбільше цілочисельне значення, менше або дорівнює кількості
SIN Повертає синус числа
SQRT Повертає квадратний корінь числа
SUM Повертає підсумкова значення виразу
TAN Повертає тангенс числа
TRUNCATE Повертає число, усеченное до певного числа десяткових розрядів

Дата функції MySQL

MySQL має ряд вбудованих функцій для роботи з датами і часом

Функція Опис
ADDDATE Повертає дату після додавання певного інтервалу часу / дати
ADDTIME Повертає час / DateTime після додавання певного часового інтервалу
CURDATE Повертає поточну дату
CURRENT_DATE Повертає поточну дату
CURRENT_TIME Повертає поточний час
CURRENT_TIMESTAMP Повертає поточну дату і час
CURTIME Повертає поточний час
DATE Витягує значення дати з виразу Date або DateTime
DATEDIFF Повертає різницю в днях між двома значеннями дати
DATE_ADD Повертає дату після додавання певного інтервалу часу / дати
DATE_FORMAT Форматує дату, зазначену в масці форматування
DAY Повертає частину дня значення дати
DAYNAME Повертає ім'я дня тижня для дати
DAYOFMONTH Повертає частину дня значення дати
DAYOFMONTH Повертає частину дня значення дати
