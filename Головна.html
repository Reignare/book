Що таке база даних?

База даних (БД) відповідно до структури даних для організації, зберігання і управління сховищ даних. Кожна база даних має один або більше різних API використовується для створення, доступу, управління, пошуку і скопіювати збережені дані.

Бази даних Mysql

MySQL є реляційною системи управління базами даних, розроблена шведською компанією MySQL АВ, тепер частина корпорації Oracle. MySQL є реляційної системи управління базами даних, реляційна база даних для зберігання даних в окремих таблицях, а не покласти всі дані в великий склад, що збільшує швидкість і підвищену гнучкість

  • Mysql є відкритим вихідним кодом, так що вам не потрібно платити додаткові збори.
  • Mysql підтримка великих баз даних. Ви можете мати справу з десятками мільйонів записів у великій базі даних.
  • Дані MySQL, використовуючи стандартну форму мови SQL.
  • Mysql дозволяє на декількох системах, а також підтримує кілька мов. Ці мови програмування, включаючи C, C ++, Python, Java, Perl, PHP, Eiffel, Ruby і Tcl.
  • Mysql має хорошу підтримку PHP, PHP в даний час є найпопулярнішою мовою веб-розробки.
  • MySQL підтримує великі бази даних, підтримка 50 мільйонів записів сховища даних, 32-розрядна файлова система таблиці максимальна підтримка 4 Гб, 64-бітові системи підтримують найбільший файл таблиці 8 Тб.
  • Mysql можуть бути налаштовані за допомогою угоди GPL, ви можете модифікувати вихідний код, щоб розвивати свою власну систему Mysql.
